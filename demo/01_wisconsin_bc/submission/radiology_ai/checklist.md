# Submission Checklist: Radiology: Artificial Intelligence

## Pre-Submission Items

- [ ] Title page with all author names, degrees, affiliations, ORCID IDs
- [ ] Structured abstract (Purpose, Materials and Methods, Results, Conclusion)
- [ ] STARD 2015 checklist completed and uploaded
- [ ] All figures at 300 DPI minimum, TIFF or high-resolution PNG
- [ ] Figure legends on a separate page
- [ ] Tables formatted per journal style (no vertical rules)
- [ ] References in Vancouver (numbered) format
- [ ] Word count within limit (original research: 4,000 words)
- [ ] Data availability statement included
- [ ] Ethics/IRB statement included
- [ ] AI writing disclosure in Methods and Acknowledgments
- [ ] Key Points (3 bullet points, max 75 characters each)
- [ ] Cover letter with suggested reviewers (2-3)
- [ ] COI disclosure forms for all authors (ICMJE format)
- [ ] ORCID IDs for all authors

## RSNA-Specific Requirements

- [ ] No AI-generated images
- [ ] LLM use disclosed as language editing only
- [ ] Supplemental materials labeled as "Appendix E1", "Figure E1", etc.
