# Artifact-Coverage Fixture (synthetic)

A privacy-free synthetic manuscript fragment with a flat Methods and Results
section, used only to exercise the Methods->Results coverage check. The demo
manuscripts place Results content under sub-headings, which prevents the forward
check from isolating Results; this fixture provides a clean, isolable Results
section so PROMISED_ABSENT can be attributed unambiguously.

## Methods

We fit a multivariable logistic regression model for the primary outcome. A
sensitivity analysis excluding participants with missing covariates was
performed to assess robustness.

## Results

The primary model showed a positive association. The sensitivity analysis
excluding participants with missing covariates was consistent with the primary
result.
