# Reviewer draft — Major comments

**3.1 The screening step is not characterised**

Suggested revisions:
- Propagate the measured false-negative rates to the screened population and report the expected number of affected cases remaining in the final cohort.
- Bootstrap confidence intervals for the reported percentiles.
- Have a second reader independently label a subsample and report agreement.
- Compare the analysed subset with the full cohort and give P values for each characteristic.

**3.2 The reference distribution**

Suggested revisions:
- State the software version used.
- Report the per-stratum n in every reference table.
- Consider modelling the outcome as a continuous function of age rather than binning by decade.
