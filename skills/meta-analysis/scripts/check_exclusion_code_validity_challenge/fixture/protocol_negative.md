# Protocol — Comparator-Restricted Systematic Review (registered)

## Eligibility criteria

Included study designs: randomised controlled trials and comparative cohort studies
were eligible. A concurrent comparator group was required for inclusion.

Single-arm and non-comparative studies were excluded (code F2 below), as were
studies of a wrong target population.

## Exclusion-code legend

| Code | Reason |
|---|---|
| F1 | Duplicate publication |
| F2 | No comparative data (single-arm / uncontrolled) |
| F3 | Full-text unavailable |
| F4 | Wrong target population |
| F5 | Insufficient sample or follow-up |
